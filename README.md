HTS Help File
======
> Below are the steps and general syntax on how to run the Howell Test Software. 

MSG command
------
> Msg command is to Give instructions to the user on how to proceed with the test Procedure
> msg command
```
msg,Power,Power OFF
```
ACK command
------
> Msg command is to Give instructions to the user on how to proceed with the test Procedure
> msg command
```
msg,Power,Power OFF
```
